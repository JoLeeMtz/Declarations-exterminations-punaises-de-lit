get /declarations?du={date_debut}&au={date_fin}

Affiche les déclarations qui ont les dates de déclaration entre les dates spécifiées dans les paramètres "du" et "au".

  • Request
  • Response

URI Parameters

  • date_debut: required(string)
  • date_fin: required(string)

Query Parameters

  • date_debut: required(date)

    Date du début de traitement. En format ISO 8601 (ex:2018-05-08).

  • date_fin: required(date)

    Date de fin de traitement. En format ISO 8601 (ex:2020-05-15).

Type: array of object

Items: ExistingDeclarations

  • NBR_EXTERMIN: (number)
  • DATE_DEBUTTRAIT: (date-only)
  • DATE_FINTRAIT: (date-only)
  • No_QR: required(string)
  • NOM_QR: required(string)
  • NOM_ARROND: required(string)
  • COORD_X: required(number)
